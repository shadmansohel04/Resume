Projects

LeetBot

LeetBot is an online multiplayer PvP game built around player profiles based on LeetCode stats. Created during Ignitian Hacks 2024, where it won first place, the game features a Python Flask backend with Socket.io for real-time interactions and a React.js frontend styled with NES.CSS. We integrated Mistral AI for unique usernames and used the LeetCode GraphQL API to track and display player stats. After the hackathon, we refined the code and published LeetBot online. Grab a friend and play!

RateTO

RateTO is a fullstack project I developed to provide Torontonians with relevant and recent data based on their location. Using OpenData Toronto's databases, RateTO offers crime scores, park scores, and transportation scores tailored to each neighborhood. For the backend, I used Python and Flask, while the frontend is built with ReactJS. This project is essentially my love letter to Toronto, where I go to school and have created many cherished memories growing up in the GTA.

TMRUN

TMRUN is a web application designed to offer fresh, niche metrics for your recent runs and running trends, encouraging runners of all levels. Unlike traditional platforms that emphasize increasing pace and reducing overall time, TMRUN introduces a broader perspective by considering factors such as consistency, elevation, and more. This approach ensures that every runner can track their progress and celebrate their achievements. Built with Node.js, Express, and MongoDB for the backend, and React for the frontend, TMRUN is hosted on Render and integrates seamlessly with Strava's API to provide accurate and up-to-date scores, along with a global leaderboard for a motivating community experience.

Arithmetic Logic Unit

I embarked on designing an Arithmetic Logic Unit (ALU) capable of managing arithmetic and logical functions on two 8-bit inputs. This endeavor demanded a deep understanding of Finite State Machines (FSM), decoders, storage units, VHDL code, among other components. This journey not only deepened my hardware expertise but also afforded me a holistic perspective as both a software and hardware engineer. Delving into the intricacies of CPUs expanded my appreciation for the interconnected nature of software and hardware in crafting fundamental computing elements.

Good Morning Script

I developed a Python script that automates daily text messages with location based weather forecasts and commute times. The script integrates with a weather API to retrieve real time forecasts and provides a brief weather description. Additionally, it calculates daily commute times using mapping services based on current traffic conditions. The system is scheduled to run every morning and sends personalized updates via SMS using the Twilio, GoogleMaps and OpenWeather APIs, improving daily planning and productivity.
